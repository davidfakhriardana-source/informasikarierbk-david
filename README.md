# Dunia Karier

## Introduction
Dunia Karier is a platform designed to help individuals navigate their career paths effectively.

## What is RIASEC
The RIASEC model categorizes careers into six personality types:
- **Realistic**: Hands-on, mechanical, and tangible roles.
- **Investigative**: Analytical and curious about the world, like scientists.
- **Artistic**: Creative and expressive, involved in arts or media.
- **Social**: Helping others and creating nurturing environments.
- **Enterprising**: Leadership and project management roles.
- **Conventional**: Organized and detail-oriented jobs, often in administration.

## Services Offered
- Career counseling
- Personality assessments
- Resume and interview preparation
- Networking opportunities

## How to Use the Website
Visit the homepage, select a service, and follow the prompts to get started.

## Key Features
- User-friendly interface
- Personalized career recommendations
- Access to expert counselors

## Purpose of Development
This website was developed to provide essential resources and guidance to students and job seekers in their career journey.

## Technologies Used
- HTML, CSS, JavaScript for frontend
- Node.js and Express.js for backend
- MongoDB for database management

## Conclusion
Dunia Karier aims to empower users through effective career guidance and resources.
